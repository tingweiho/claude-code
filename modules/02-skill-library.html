The Skill Library

Load-on-demand expertise — a team of on-call specialists who show up only when the job fits.

I never page a specialist by name

I never type "use the SQL skill." I describe what I want, and the right expertise loads itself. A skill is just a folder containing a SKILL.md file and a one-line description that decides when it fires.

company-data-stack/ One skill = one folder
SKILL.md The full instructions + the trigger description at the top
💡
The business card on the desk

At startup Claude reads only the descriptions — the specialist's business card is always on the desk. The full SKILL.md walks in only when the job fits, keeping Claude's context window lean.

The description IS the trigger

Open any SKILL.md and the top few lines tell the whole story. Here's the frontmatter of the data-viz skill.

CODE 
---
name: data-viz
description: Use whenever making a chart, plot, or
  visualization in a notebook. Trigger on "plot this",
  "visualize", "show the distribution", "compare segments".
---
PLAIN ENGLISH

Every skill starts with a name and a description.

The name is just the folder's label.

Only this description sits in Claude's memory until needed.

Those example phrases are the trip-wires — say something close and the full skill loads.

ℹ️
Writing a good description is its own craft

Too vague and the skill fires constantly; too narrow and it never fires. There's even a prompt-authoring skill for tuning exactly this — and yes, a notebook is the data-analyst's working canvas.

Where the skills came from

None of this was built from zero. The library is three origins braided together.

🛠️

Built from scratch

The company-specific skills encode internal know-how no download could carry — warehouse conventions, the canvas rules, the house doc style.

🍴

Forked & rebranded

codebase-to-course and the frontend-slides engine behind company-slides come from open-source builder Zara Zhang (github.com/zarazhangrui). I forked them and swapped in my own branding and conventions.

📦

Straight from Anthropic

skill-creator and pdf are Anthropic's own skills; the ralph-loop plugin installs from the official anthropics/claude-plugins-official marketplace.

💡
Don't build from zero

Fork a good skill from a repo, rebrand it, and bend it to your conventions — then use skill-creator to grow your own. It's one of the best-practice threads we pull together at the end.
