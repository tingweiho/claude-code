Claude Code 實戰手冊

一位資深資料科學家實際的每日工作方式：設定、技能、自動化與記憶 — 一趟誠實的導覽。

嗨 — 我是 Ting-Wei。Claude Code 已經悄悄成為我工作的骨幹：寫 SQL、發布程式碼、畫圖表、草擬更新，並在我睡覺時把那些無聊的事跑完。

這個頁面是那套設定的導覽 — 不是一本說明書，只是那些真正掙得一席之地的零件，以及幾個讓我付出代價才學到的教訓。

開始前先說一句：這只是我的工作方式 — 它不一定是最佳實踐。我盡量讓自己一致地遵循它，並透過自學、反覆試誤持續打磨。所以請把這當作想法與經驗分享，而不是一套規則手冊。
